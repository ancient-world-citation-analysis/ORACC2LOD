ORACC2LOD JupyterBook#

This work includes a complete methodology and implementation for obtaining the full dataset of texts in transliteration and catalog metadata from ORACC: the Open Richly Annotated Cuneiform Corpus. The code explored in the book sections were inspired by the work of Niek Veldhuis, professor of Assyriology at UC Berkeley. In particular, his work on COMPASS enabled the parsing of the ORACC text data from json files. It is necessary to cite the specialists' project who provided the English translations, and as such we include the references to each ORACC project and these specialists in our repository with links to the ORACC Project List.

The following JupyterBook is build for transparency, using the FAIR principles, and is intended to illustrate the methodology for reproducibly migrating ORACC data to Linked Open Data in FactGrid, using Python Jupyter Notebooks. These notebooks cover processes such as data downloading, extraction, and analysis, from initial ZIP files processing in 1-ORACC_ZIPS.ipynb to advanced data cataloging and linguistic analysis in 4-Megacatalogue.ipynb and 8-ORACC Projects by Language.ipynb. Each notebook is designed for specific data processing tasks, forming a comprehensive workflow for the necessary data formatting of the cuneiform records.
